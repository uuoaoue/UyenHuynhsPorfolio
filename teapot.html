The Teapot

2021
The Teapot project bases on the origin story of The Utah teapot. The Utah teapot, or the Newell teapot, is a 3D test model that has become a standard reference object within the computer graphics community. The teapot model was created in 1975 by early computer graphics researcher Martin Newell, a member of the pioneering graphics program at the University of Utah. This project consists of two part:

Part 1: Create a basic teapot using the fundamentals of creating radial symmetry along profiles, rails and 2d extrusions (use revolve,loft and/or sweep).
Part 2: Design a unique model based on a 2d handmade sketch using the same techniques of using (revolve,sweep,loft).
